## KEY MESSAGING ELEMENTS

### Core Message (Must Include):
- **What it is:** AI-Powered Planning & Creation Studio
- **Key benefit:** Plan, create, organize, post - all with AI / Prevents burnout
- **Main features:** AI Strategy, AI Captions, AI Organization, Manual Posting
- **Workflow:** Plan with AI → Create with AI → Organize with AI → Post Manually

### Essential Components:
1. **Problem** - Creator burnout, overwhelming planning
2. **Solution** - EchoFlux.ai (AI-powered planning studio)
3. **Features** - AI strategy, AI captions, AI organization, manual posting
4. **Benefit** - Prevent burnout, stay in control, plan smarter with AI
5. **Workflow** - Plan, create, organize with AI, then post manually (no auto-posting mentioned once)
6. **Brand** - EchoFlux.ai name and website

### Word Count Target:
- **15 seconds = ~40-45 words** at normal speaking pace
- **Fast-paced = ~50-55 words** (energetic delivery)
- **Measured = ~35-40 words** (clear, confident delivery)

## PRODUCTION NOTES

### Visual Style:
- **Ultra-Fast Cuts:** 1-2 seconds per shot maximum
- **High Energy:** Dynamic animations, smooth transitions
- **Product-Focused:** Show actual app interface
- **Brand Consistency:** EchoFlux.ai colors and styling
- **Mobile-Optimized:** Text must be readable on small screens

### Audio:
- **Music:** Upbeat, modern, tech-forward (instrumental, no vocals)
- **Voice:** Clear, confident, energetic (professional voiceover)
- **Pacing:** Fast but clear - every word counts
- **Tone:** Confident, solution-focused, empowering

### Text Overlays:
- **Font:** Bold, modern sans-serif (e.g., Inter Bold, Poppins Bold)
- **Size:** Extra large for mobile viewing
- **Color:** High contrast, brand colors
- **Animation:** Quick fade-ins, no slow transitions
- **Duration:** Text appears for 2-3 seconds max

### Timing Breakdown:
- **0:00-0:04** (4 sec): Problem hook
- **0:04-0:11** (7 sec): Solution and features
- **0:11-0:14** (3 sec): Value proposition
- **0:14-0:15** (1 sec): Brand close

### Call-to-Action:
- **Primary:** Website URL (echoflux.ai)
- **Secondary:** "Get Started" or "Try Free"
- **Visual:** Prominent, readable, clickable

## OFFLINE USAGE NOTES

This script is designed for offline production. All visual references are based on the actual EchoFlux.ai application interface. When recording:

1. **Use actual app screenshots/recordings** from the live application
2. **Maintain brand consistency** with EchoFlux.ai colors and styling
3. **Test on multiple devices** - especially mobile (most common viewing)
4. **Create multiple versions** - different tones, different feature focuses
5. **Include captions/subtitles** - many viewers watch without sound
6. **Optimize file size** - for fast loading on social platforms
7. **Create square and vertical versions** - for different platform requirements
