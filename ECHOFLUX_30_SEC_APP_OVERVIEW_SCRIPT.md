## KEY MESSAGING POINTS

### Primary Message:
"EchoFlux.ai is an AI-powered planning and creation studio. Plan with AI, create with AI, organize with AI, then post manually. No auto-posting - you stay in control and prevent burnout."

### Core Value Propositions:
1. **AI-Powered Everything** - Plan, create, organize - all with AI
2. **Prevents Burnout** - Reduces overwhelm, keeps creators creating
3. **You Stay in Control** - Manual posting, no auto-posting
4. **Complete Workflow** - Plan → Create → Organize → Post
5. **All-in-One** - Replaces multiple tools with one platform
6. **Offline Planning** - No social account connections needed

### Features to Emphasize:
- AI Content Strategist (multi-week roadmaps)
- AI Caption Generation
- AI Calendar Organization
- Manual Posting (copy content out to post)
- Burnout Prevention (organized, stress-free planning)

## PRODUCTION NOTES

### Visual Style:
- **Modern & Clean:** Minimalist UI, bright colors, smooth animations
- **Fast-Paced:** Quick cuts, dynamic transitions
- **Product-Focused:** Show actual app interface, not just graphics
- **Professional:** High-quality screen recordings, polished animations

### Audio:
- **Music:** Upbeat, modern, tech-forward background music (instrumental)
- **Voice:** Clear, confident, energetic (male or female, professional voiceover)
- **Pacing:** Fast but clear - approximately 150-160 words for 30 seconds

### Text Overlays:
- **Font:** Modern, sans-serif (e.g., Inter, Poppins)
- **Size:** Large enough to read on mobile
- **Color:** High contrast, brand colors
- **Animation:** Smooth fade-ins, slide-ins

### Call-to-Action:
- **Primary CTA:** "Start Planning Today" or "Get Started Free"
- **Secondary:** Website URL (echoflux.ai)
- **Visual:** Prominent button or text link
